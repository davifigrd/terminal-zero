## 📌 SOBRE O PROJETO

O Terminal Zero é um jogo 2D do gênero puzzle desenvolvido como atividade acadêmica da disciplina de Raciocínio Lógico, no curso de Análise e Desenvolvimento de Sistemas (ADS) da FAMINAS, ministrada pelo professor Nícollas Cretton.

O jogo simula uma experiência de ambiente digital/terminal, onde o jogador precisa resolver desafios lógicos para avançar em diferentes níveis.

---

## 🧠 OBJETIVO
```
>>> OBJETIVO: HACKEAR O SISTEMA
>>> MÉTODO: RESOLVER PUZZLES LÓGICOS
>>> PROGREÇÃO: AUMENTANDO A COMPLEXIDADE
>>> FALHA DE ESTADO: REINICIANDO...
```
O jogador deve resolver uma sequência de puzzles lógicos que aumentam gradualmente em dificuldade, exigindo raciocínio, análise e tomada de decisão para avançar.
