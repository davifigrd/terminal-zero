🕹️ FASES:
1. Sistema bloqueado:
Ideia: o jogador precisa ativar o sistema corretamente
Tela com 3–4 botões (ex: Energia, Rede, Terminal, Segurança)
Só existe uma ordem correta simples (ex: Energia → Rede → Terminal)
Se errar, reinicia

dicas energia, rede, terminal

2. Encontrar arquivo correto:
- Vários arquivos na tela
- Apenas um é o correto
- Clique simples
- Sem raciocínio pesado

3. Código numérico:
- Inserir código (3 ou 4 números)
- Dica simples na tela
- Primeira interação com input
- Ainda bem guiado

4. Ordem correta:
- Começa a exigir memória
- Um pouco mais difícil

5. Senha com lógica:
- Mais interpretação
- Já não é tão óbvio quanto antes

6. Conectar pontos:
- Introduz coordenação (drag)
- Um pouco mais complexo mecanicamente

7. Sistema com erro:
- Jogador precisa identificar problemas
- Menos direto, mais atenção

8. Puzzle com tempo:
- Pressão + execução
- Aumenta dificuldade mesmo sendo mecânica conhecida

9. Combinação de mecânicas:
- Junta tudo (ex: código + ordem)
- Teste final antes do boss

9,5. fase de labirinto com o mouse de susto

10. Chefão (vírus)
- Pressão + múltiplas ações
- Dinâmico
- Mais difícil do jogo
